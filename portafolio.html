Project 1: Flor de Lis

A multi-page website that simulates a digital ordering system.
-The main page introduces the product with a clean and visually appealing layout.

-The second page dynamically displays all available products using data from a JSON file, organized in card-based sections with interactive buttons for user selection.
-The order page allows users to submit their request, capturing their input and sending the information directly via WhatsApp. Developed using HTML, CSS, and JavaScript, with a focus on user interaction, dynamic content rendering, and functional flow.
